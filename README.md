- Преподаватели могут добавить личные события;
- Удалить личные события;
- Редактировать личные события;<p>

<p>Тестирование API<p>
								
<p>Регрессионное тестирование:

- Регрессионый чек-лист</p>

<p>Smoke и Приёмочное тестирование:
  
- Тест-кейсы qase.io;
